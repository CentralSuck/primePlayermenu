Installation:

1) Put the resource in your resource folder
2) run the items.sql in your database
3) Setup the config.lua
4) Add in your server.cfg:
	ensure NativeUI_Reloaded
	ensure jsfour-idcard
	ensure primePlayermenu

Depedencies:
- ESX
- NativeUI_Reloaded
- jsfour-idcard
